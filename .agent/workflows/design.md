## UI Design Improvement

Read the `ui-designer` skill and follow its instructions:

1. Read the skill file (`view_file`): `.agent/skills/ui-designer/SKILL.md`

1. Start dev server:

```bash
npm run dev
```

1. Open the specified page in the browser and take a screenshot of the current state.

2. Identify design issues:
   - Color mismatches
   - Typography inconsistencies
   - Layout problems
   - Responsive design gaps
   - Missing animations

3. Apply improvements.

4. Take a screenshot after changes and compare.

5. Lint and type-check:

```bash
npm run lint
```

1. Summarize results with a before-and-after comparison.
